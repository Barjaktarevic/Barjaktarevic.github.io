Na Filozofskom fakultetu u Osijeku diplomirao sam kao magistar hrvatskog i engleskog jezika i književnosti. Tijekom studija bio sam dobitnik dekanove nagrade i dviju najviših pohvala (summa cum laude).

Već osam godina radim kao nastavnik Engleskog jezika (Gimnazija Vukovar). Hobiji su mi učenje stranih jezika, izrada predmeta od drveta i igranje video igara.

Porijeklom sam iz Vukovara, ali živim u Osijeku. Sretno sam oženjen i imam jednog sina.

Frontend

HTML i CSS (uključujući Flexbox, Grid, responsive design, accessibility). Radio sam s nekoliko CSS frameworks (ponajprije Bootstrap, zatim Tailwind i ponešto Material UI uglavnom u React projektima).
Dobro vladam modernim JS-om (uključujući OOP), a izradio sam i nekoliko manjih projekata u TS-u. Znam osnove jQuery-ja, ali se dosad njime nisam služio na nekom ozbiljnijem projektu.
Od frontend frameworka radim samo u vanilla Reactu u kojem znam upotrijebiti hooks (useState, useEffect, useRef, useReducer, useContext) te imam kakvo-takvo poznavanje class-based komponenti. Također znam izraditi custom hooks te upotrijebiti React zajedno s Firebaseom.

Relevantni certifikati: Web Dev Bootcamp 2022

Backend

Svakodnevno koristim NodeJS + Express, iako sam izradio i neke manje projektiće u vanilla Nodeu. Znam raditi HTML sanitizaciju, error handling, MVC pristup te koristim neke relativno ustaljene Node pakete (morgan, dotenv, helmet, JOI, mongoose itd.). Što se tiče Node templatinga služim se samo EJS-om sa željom da u dogledno vrijeme naučim i Pug.
Pored Nodea znam se služiti i PHP-jem na osnovnoj razini te sam u jednom projektu koristio i Laravel framework.
Također imam rudimentarno znanje Pythona i cilj da u budućnosti savladam Django i Flask.

Relevantni certifikati: Web Dev Bootcamp 2022

Baze podataka

Najbolje se znam služiti MongoDB + mongoose frameworkom, iako sam s Mongom jedno vrijeme radio i bez mongoosea. Svakodnevno upotrebljavam MongoDBCompass za rad na svojim projektima, te se također služim i MongoAtlasom. Sposoban sam raditi CRUD operacije bez većih problema.
Pored Monga znam se služiti i MySQL-om (također jedan položen tečaj na Udemyju) u kojem također mogu vršiti CRUD operacije, a ponekad me možete naći i kako rješavam SQL query probleme na raznim stranicama.
Konačno, znam se služiti Firebaseom - u kontekstu React projekata - uključujući i postavljanje Firebase pravila, ali imam još dosta prostora za napredak (pogotovo u odnosu na Firebase 9).

Relevantni certifikati: Web Dev Bootcamp 2022    •   MySQL Botcamp

DSA

Znam riješiti početničke probleme koji se mogu pronaći na Leetcodeu i CodeWars (na kojem sam rank 5). Trenutno slušam jedan Udemy tečaj na ovu temu (Colt Steele's Data Structures and Algorithms) te također slušam i radim zadatke za CS50. Svjestan sam da je ovo područje na kojem ću trebati puno više raditi u budućnosti, ali isto tako da bi sad bio gubitak vremena previše se posvetiti apstrakciji dok moram predstaviti konkretne rezultate.

Ostalo

Znam se služiti Gitom (također jedan položen tečaj na Udemyju) te u nešto manju ruku Githubom jer dosad nisam imao previše potrebe dijeliti svoj kod s drugima. Nadam se da ću uskoro moći raditi na kakvim open source projektima pa da ću usavršiti svoje znanje u tom području.
Također se služim Git Bashom i Powershellom na Windowsima i bez problema upotrebljavam osnovne komande (cd, ls, rm, mkdir, mv...).
Znam deployati projekt na Github pages, Firebase hosting te na neke od alternativa za Heroku poput Railway, Cyclic ili Render.
Uz to pokušavam čitati koliko god mogu dokumentacije te pratim YouTube kanale: Traversy Media, Web Dev Simplified, Kevin Powell, the Net Ninja, freeCodeCamp (na čijoj sam stranici položio Responsive Design tečaj i JavaScript tečaj) kako bih proširio svoje znanje.

Relevantni certifikati: Git & Github Bootcamp
